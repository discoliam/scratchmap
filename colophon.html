How it all works

The map is powered by jVectorMap, and SVG Mapping plugin.

The Country List was based on the GDP Tutorial, but updated somewhat by myself with the missing regions.

The country specific data is called from the Rest Countries API, based on ISO 3166-1 alpha-2 country codes. We use this to retrieve Name, Currency, Population and Timezone information.

Parse is used as a database to store the visited countries, using their Javascript SDK to do all the work.

The calculations for countries visited are based on the data from the database. The percentage is rounded to the nearest whole figure. The pie-chart is powered by the Chart.js library.

The front end is powered by Bootstrap, with the Flatly theme from Bootswatch added over the top, and Font Awesome for icons. These all delivered by the Bootstrap CDN.

Scratch Map was designed and built buy Liam Richardson, with technical help from Tom Burton.
